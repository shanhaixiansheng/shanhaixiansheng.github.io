# 工业机器人查询助手

这是一个工业机器人报警代码和系统变量查询系统的项目。

## 功能特点

1. 支持主流工业品牌（FANUC、KUKA、ABB、Yaskawa）的报警代码和系统变量查询
2. 支持精确匹配和模糊搜索功能
3. 提供管理员界面，查看数据统计和网站访问统计

## 管理员密码

为了安全起见，管理员密码不直接在代码中存储。管理员密码可以在 `app.js` 中的 `validPasswords` 数组中设置。

当前项目使用示例密码数组，实际部署时应修改为更安全的验证方式。

## 部署说明

1. 修改 `app.js` 中的密码验证逻辑，使用更安全的方式存储和验证密码
2. 可以考虑使用环境变量、服务器端验证或其他安全机制
3. 确保密码信息不会被意外提交到公共代码仓库

## 项目结构

- `index.html` - 主页面
- `app.js` - 主要JavaScript逻辑
- `styles.css` - 样式文件
- `data/` - 数据文件夹，包含各品牌的报警代码和系统变量数据

## 注意事项

本项目仅用于工业机器人技术参考，实际应用请参考机器人厂商提供的官方文档。
